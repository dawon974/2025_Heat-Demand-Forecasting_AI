## 🚀 앞으로 보완할 점 및 느낀 점
📈 이번 프로젝트: 열수요 예측을 통한 모델 성능 개선
이번 열수요 예측 프로젝트에서는 시계열 기반 예측 모델과 머신러닝/딥러닝 모델을 폭넓게 적용하며, 정량적 평가 지표(MAPE)를 중심으로 월별 성능을 비교 분석하였습니다.

RandomForest 모델은 전월에 걸쳐 가장 안정적인 예측 성능을 보이며, 실시간 대응이 필요한 환경에 적합함을 확인했습니다.

LSTM + XGBoost 앙상블 모델은 추세 예측에서 우수한 결과를 나타냈으며, 특히 급격한 변화가 있는 시간대에서도 안정적인 결과를 도출했습니다.

반면, GRU와 RNN 모델은 계절성 및 피크 시간대 예측 정확도에 한계를 보여, 모델 구조의 개선 여지가 있음을 확인했습니다.

또한, 단순 예측 정확도 비교에 그치지 않고 실제 운영 적용 가능성, 확장 가능한 feature 구성, 계절성과 외부 요인의 반영 필요성 등 실무적 관점에서의 해석까지 확장하였습니다.

💡 느낀 점 및 보완 방향

이번 프로젝트를 통해 머신러닝과 시계열 분석 기법 각각의 특성과 한계를 비교하며, 목적에 맞는 모델 선택이 핵심임을 체감했습니다.
또한, 데이터의 계절성, 시간 지연 효과, 외부 변수(기온, 공휴일 등)의 중요성을 실무 적용 관점에서 재확인할 수 있었습니다.

향후에는 다음과 같은 보완이 필요합니다:

계절성 반영 강화: 월, 계절, 공휴일 여부 등의 파생 변수 추가

입력 변수 확장: 공급압력, 체감온도, 습도 등 다양한 외부 요인 포함

딥러닝 모델 구조 개선: Attention 도입, sequence length 조정 등

이번 경험은 단순히 모델 성능을 비교하는 수준을 넘어서, 데이터 기반으로 실질적인 운영 효율화 방안을 설계하고, 예측 데이터를 비즈니스 가치로 전환하는 실전 감각을 키우는 데 중요한 기회가 되었습니다.
